# Stress_Agent_Adaptive-Support
Edge-AI Based Real-Time Mental Health Monitoring System
📌 Overview

This repository presents an AI-driven, real-time mental health monitoring system that performs multi-class sentiment analysis on psychological text data and deploys optimized models on edge devices for efficient inference.

The system integrates:

Natural Language Processing (NLP)
Deep Learning architectures (CNN, CNN-LSTM)
Explainable AI (XAI)
Edge deployment using TensorFlow Lite on Raspberry Pi
